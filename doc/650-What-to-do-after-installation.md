
new user:

* use it as-is for a while. Avoid getting caught up with "now I need to harden my Qubes"
* Work out how you will back up your data, when you start using it for real
* Learn how to use disposables
     * Exercise: make a new disposable template which has no network access
*  Learn how to attach a USB key to a qube
   * Learn the difference between attaching a device and attaching a partition
   * ...


When getting serious:

* set a password in Dom0
     * it may be helpful if there is ever a boot problem
* Start doing those backups
